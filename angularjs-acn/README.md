# AngularJS ACN validator

AngularJS 1.x implementation of the ACN validator (`src/`). Repository-wide overview, run instructions for **both** Angular and AngularJS apps, and tutorial folders are documented in **[`../README.md`](../README.md)**.

```bash
npm install
npx --yes http-server -p 8080 -c-1
```

Then open **http://localhost:8080/src/**

```bash
npm test
```
